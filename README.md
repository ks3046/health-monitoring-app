# Health Monitoring & Patient Management App

An Android application designed to help users track health metrics, detect anomalies, and provide basic health insights.

---

## Problem
Users often struggle to consistently track health data and identify early warning signs of health issues.

---

## Solution
A simple and accessible mobile app that:
- Logs user health data
- Detects abnormal patterns
- Provides basic suggestions
- Enables continuous monitoring

---

## Target Users
- Individuals tracking daily health metrics
- Users in low-access healthcare environments
- Non-technical users needing simple interfaces

---

## Features
- Health data logging (symptoms, metrics)
- Anomaly detection (rule-based)
- Basic health suggestions
- Simple and intuitive UI

---

## Product Thinking
- Focused on reducing user friction in data entry
- Designed for accessibility and ease of use
- Considered real-world usability and edge cases

---

## Tech Stack
- Android Studio
- Kotlin
- Firebase

---

## Future Improvements
- AI-based health predictions
- Doctor integration
